Teste de Leitor de Código de Barras

Esta ferramenta ajuda você a verificar se o leitor de código de barras está funcionando corretamente.

⚠️ Use um leitor físico. Passe o código e pressione Enter.

Teste Simulado (sem leitor físico)

Como verificar se o leitor está funcionando:

  • Pressione as teclas de teste acima para simular a leitura de um código de barras
  • Se os campos ficarem verdes, significa que o sistema está funcionando
  • Se você tem um leitor físico, aproxime e escaneie um código de barras real
  • Observe se o código aparece no campo de entrada e o status é atualizado
  • Verifique se o leitor está em modo de interface de teclado (HID)
  • Teste pressionando "Enter" após a leitura para verificar o comportamento completo
